QUICK SETUP GUIDE
=================

1. INSTALL DEPENDENCIES
   ----------------------
   pip install -r requirements.txt

2. CREATE .ENV FILE
   -----------------
   Copy .env.example to .env and add your API keys:
   
   OPENAI_API_KEY=sk-proj-your-key-here
   ANTHROPIC_API_KEY=sk-ant-your-key-here
   HUGGINGFACE_TOKEN=hf_your-token-here
   PORT=5000

   NOTE: You need at least ONE of these API keys:
   - OPENAI_API_KEY (Primary - recommended)
   - ANTHROPIC_API_KEY (Fallback)
   - PROJECT_ID + gcloud auth (Gemini - secondary)

3. GET API KEYS
   ------------
   OpenAI: https://platform.openai.com/api-keys
   Anthropic: https://console.anthropic.com/settings/keys
   Hugging Face: https://huggingface.co/settings/tokens

4. RUN THE APPLICATION
   --------------------
   python app.py

   Then open: http://localhost:5000

5. VERIFY SETUP
   -------------
   python test_setup.py

6. QUICK TEST
   -----------
   Try these inputs in the web interface:
   
   - YouTube: https://www.youtube.com/watch?v=dQw4w9WgXcQ
   - Topic: "trending AI developments today"
   - Prompt: "create a post about machine learning"

TROUBLESHOOTING:
----------------
Port 8080 blocked? Change PORT in .env to 5000
API errors? Check your API keys in .env
Import errors? Run: pip install -r requirements.txt
